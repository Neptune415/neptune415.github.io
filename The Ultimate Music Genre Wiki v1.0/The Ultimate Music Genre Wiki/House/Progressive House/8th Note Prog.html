8th Note Prog

Timeline：最早2006年并开始发展，2008年开始流行至今

上位：Progressive House、McProg、Trance、Electro House

下位：无

A.K.A. Dut dut dut dut、Dut dut dut dut dut dut dut dut

八分音符渐进（8th Note Prog）始于2006年，以典型的McProg风格的结构为基础，采用了一组非常基础的八分音符（因此得名），并大量运用了包络、滤波器和失谐振荡器，使其从紧凑封闭的“dit dit dit dit”过渡到开放模糊的“dat dat dat dat”，但大部分时间听起来就像“dut dut dut dut”，这也是它的别称之一。

8th Note Prog在Progressive House的历史上取得了巨大成功，但实际上它并未产生历史性的深远影响或重要意义，唯一例外的是，8th Note Prog的演示模板连同其核心元素的采样和循环片段被收录到了最常见的宿主软件之一FL Studio（原名Fruityloops）中。许多业余音乐人无法抗拒免费采样和采样包的诱惑，这些采样包能为他们省去大量工作，因此他们纷纷涌向制作自己的8th Note Prog音乐。

与此同时，社交媒体正蓬勃发展，一个名为Beatport的新MP3销售网站应运而生，这引发了一场大火的文化转化。突然间，不仅制作8th Note Prog变得异常容易，上传、分享自己的音乐作品也变得轻而易举，在接下来的几年里，Progressive House音乐界几乎只听到这种风格。8th Note Prog整个流派的形成被很多人认为是源于懒惰、缺乏专业素养的音乐创作，以及音乐潮流仅能概括为一两个核心元素的瓶颈化。8th Note Prog的浪潮并未持续太久，因为这种流派在范围和原创性上过于局限，注定无法持久，到了2010年，所有人都开始对它感到厌倦。
